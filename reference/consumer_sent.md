# Consumer Sentiment Data

University of Michigan Consumer Sentiment Data, FRED code UMCSENT

## Usage

``` r
consumer_sent
```

## Format

### `consumer_sent`

A data frame with 120 rows and 3 columns

- symbol:

  FRED identifier

- date:

  Date of report

- pricce:

  Observation

## Source

<https://fred.stlouisfed.org/>
