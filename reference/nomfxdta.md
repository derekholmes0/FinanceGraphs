# Nominal FX levels

Closing Nominal currency levels (local currency per dollar)

## Usage

``` r
nomfxdta
```

## Format

### `nomfxdta`

A tibble with 13134 observations and 3 columns

- date:

  Date Of Observation

- variable:

  CUrrency

- value:

  Currency/USD
